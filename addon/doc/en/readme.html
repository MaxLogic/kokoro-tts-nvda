MaxLogic Kokoro TTS

MaxLogic Kokoro TTS is a separate Kokoro-based synthesizer for NVDA.

This development install uses a distinct internal identity from the existing kokoroTTS add-on so both can coexist.

  • The synth prefers CUDA when the helper environment exposes CUDAExecutionProvider.
  • If CUDA is not available, the helper can be provisioned for DirectML or CPU.
  • On 32-bit NVDA, GPU inference runs through the optional 64-bit helper process.
